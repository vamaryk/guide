# Landing And Discovery

## SDD-1. Просмотр лендинга

### Цель

Гость открывает сайт и получает общее представление о проекте и доступных экскурсиях.

### Предусловия

- frontend доступен
- backend доступен для загрузки каталога

### Основной сценарий

1. Гость открывает главную страницу.
2. Система показывает hero, карту, каталог, FAQ, партнёров, форму и контакты.
3. Гость использует фиксированное меню для перехода по разделам.

### Ожидаемый результат

- ключевые разделы доступны без ошибок
- навигация работает по якорям
- каталог загружается с backend

### Артефакты реализации

- `src/App.tsx`
- `src/index.css`
- `GET /api/objects`

## SDD-2. Изучение объекта через каталог

### Цель

Гость находит интересующий объект и открывает подробную информацию.

### Основной сценарий

1. Гость переходит в блок `Каталог экскурсий`.
2. Гость использует фильтры по аудитории.
3. Система показывает подходящие карточки.
4. Гость нажимает `Подробнее`.
5. Система открывает модальное окно объекта.

### Ожидаемый результат

- карточки не ломают мобильную сетку
- модальное окно содержит расширенное описание объекта
- из модального окна можно перейти к бронированию

### Артефакты реализации

- `src/App.tsx`
- `GET /api/objects`
- `GET /api/objects/:slug`

## SDD-3. Изучение объекта через карту

### Цель

Гость использует карту Саратовской области как точку входа в каталог.

### Предусловия

- задан `VITE_YANDEX_MAPS_API_KEY`

### Основной сценарий

1. Гость открывает блок карты.
2. Система отображает маркеры объектов.
3. Гость нажимает на маркер.
4. Система открывает popup с краткой информацией.
5. Гость нажимает ссылку на подробности.
6. Система открывает соответствующую карточку объекта в каталоге.

### Альтернативный сценарий

1. API-ключ не задан.
2. Система показывает fallback вместо интерактивной карты.

### Ожидаемый результат

- карта центрируется на Саратовской области
- popup связывает карту и каталог
- fallback остаётся понятным и не ломает страницу

### Артефакты реализации

- `src/components/YandexRegionMap.tsx`
- `VITE_YANDEX_MAPS_API_KEY`
